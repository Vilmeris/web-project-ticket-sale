# Event Ticket Purchasing Platform

A full-stack web application for managing and purchasing tickets for concerts, cinema, and theater performances. [cite: 1]

##Overview
This platform provides a seamless user experience for discovering events and booking tickets. [cite: 1] [cite_start]It focuses on a scalable backend architecture and a responsive frontend to handle diverse event categories and real-time transactions. [cite: 1]

##Tech Stack
* **Backend:** ASP.NET MVC, C# 
* **Database:** SQL Server (Complex relational modeling)
* **Frontend:** HTML5, CSS3, JavaScript

##Key Features
* **Multi-Category Management:** Specialized handling for cinema, theater, and concerts.
* **Relational Database:** 3rd Normal Form (3NF) compliant schema for data integrity. 
* **MVC Architecture:** Clean separation of concerns for better maintainability.
